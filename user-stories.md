## 1. Post overzicht bekijken
**Als foodblogger** wil ik een overzicht van al mijn posts zien  
**zodat** ik snel kan beheren wat er online staat.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik zie een lijst met alle posts
  - Elke post toont titel en datum
  - Er is een knop om een post te openen

---

## 2. Nieuwe post aanmaken
**Als foodblogger** wil ik een nieuwe post kunnen aanmaken  
**zodat** ik nieuwe recepten kan publiceren.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan een titel invullen
  - Ik kan inhoud invullen
  - De post wordt opgeslagen in de database

---

## 3. Post bekijken
**Als foodblogger** wil ik een post kunnen bekijken  
**zodat** ik kan controleren hoe deze eruitziet.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik zie titel en inhoud
  - De juiste post wordt geladen via ID

---

## 4. Post bewerken
**Als foodblogger** wil ik een bestaande post kunnen bewerken  
**zodat** ik fouten kan aanpassen.

- Prioriteit: must
- Acceptatiecriteria:
  - Bestaande data wordt vooraf ingevuld
  - Wijzigingen worden opgeslagen

---

## 5. Post verwijderen
**Als foodblogger** wil ik een post kunnen verwijderen  
**zodat** oude of foutieve content weg is.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan een post verwijderen
  - De post verdwijnt uit het overzicht

---

## 6. Bevestiging bij verwijderen
**Als foodblogger** wil ik een bevestiging krijgen bij verwijderen  
**zodat** ik niet per ongeluk content verwijder.

- Prioriteit: nice
- Acceptatiecriteria:
  - Er verschijnt een bevestigingsmelding
