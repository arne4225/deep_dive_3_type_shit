# User Stories – Foodblog CMS

## 1. Post overzicht bekijken
**Als foodblogger** wil ik een overzicht van al mijn posts zien  
**zodat** ik snel kan beheren wat er online staat.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik zie een lijst met alle posts
  - Elke post toont titel en datum
  - Er is een knop om een post te openen

---

## 2. Nieuwe post aanmaken
**Als foodblogger** wil ik een nieuwe post kunnen aanmaken  
**zodat** ik nieuwe recepten kan publiceren.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan een titel invullen
  - Ik kan inhoud invullen
  - De post wordt opgeslagen in de database

---

## 3. Post bekijken
**Als foodblogger** wil ik een post kunnen bekijken  
**zodat** ik kan controleren hoe deze eruitziet.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik zie titel en inhoud
  - De juiste post wordt geladen via ID

---

## 4. Post bewerken
**Als foodblogger** wil ik een bestaande post kunnen bewerken  
**zodat** ik fouten kan aanpassen.

- Prioriteit: must
- Acceptatiecriteria:
  - Bestaande data wordt vooraf ingevuld
  - Wijzigingen worden opgeslagen

---

## 5. Post verwijderen
**Als foodblogger** wil ik een post kunnen verwijderen  
**zodat** oude of foutieve content weg is.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan een post verwijderen
  - De post verdwijnt uit het overzicht

---

## 6. Bevestiging bij verwijderen
**Als foodblogger** wil ik een bevestiging krijgen bij verwijderen  
**zodat** ik niet per ongeluk content verwijder.

- Prioriteit: nice
- Acceptatiecriteria:
  - Er verschijnt een bevestigingsmelding

## 7. Account aanmaken (signup)
**Als bezoeker** wil ik een account kunnen aanmaken  
**zodat** ik posts kan publiceren.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan een gebruikersnaam en wachtwoord invullen
  - Het wachtwoord wordt veilig opgeslagen
  - Het account wordt opgeslagen in de database

---

## 8. Inloggen
**Als gebruiker** wil ik kunnen inloggen  
**zodat** ik toegang krijg tot het aanmaken van posts.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan inloggen met mijn accountgegevens
  - Bij een succesvolle login word ik herkend als gebruiker
  - Niet-ingelogde gebruikers kunnen geen posts aanmaken

---

## 9. Post met tags aanmaken
**Als ingelogde gebruiker** wil ik tags kunnen toevoegen aan een post  
**zodat** mijn content beter georganiseerd is.

- Prioriteit: must
- Acceptatiecriteria:
  - Ik kan één of meerdere tags selecteren
  - Tags worden gekoppeld aan de post
  - Bestaande tags worden hergebruikt

---

## 10. Posts zoeken en filteren
**Als foodblogger of bezoeker** wil ik posts kunnen zoeken op titel, inhoud of tags  
**zodat** ik snel relevante posts vind.

- Prioriteit: nice
- Acceptatiecriteria:
  - Zoekterm kan ingevoerd worden
  - Filter op één tag is mogelijk
  - Zoek + tag filteren tegelijk werkt
