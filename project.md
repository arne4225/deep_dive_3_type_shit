# Project: Foodblog CMS (Mini)

## Beschrijving
Een eenvoudige webapplicatie waarmee een foodblogger recepten/posts kan beheren.
De gebruiker kan posts aanmaken, bekijken, bewerken en verwijderen.

## Doelgroep
Foodbloggers of content creators die zonder technische kennis hun recepten willen beheren.

## In scope
- Overzicht van blogposts
- Nieuwe post aanmaken
- Post bewerken
- Post verwijderen
- Opslag in MySQL database

## Out of scope
- Gebruikersaccounts
- Authenticatie
- React / frontend frameworks
- Likes, reacties of tags
